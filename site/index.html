MonoGame UI, WPF mindset

Framework docs that look like a product, not a folder dump.

This is a single-page GitHub Pages mock for InkkSlinger: a WPF-inspired UI framework built for MonoGame/DesktopGL. It is intentionally lightweight, visually opinionated, and easy to replace with real documentation later.

Why this page works

Made for GitHub Pages without adding a docs toolchain.

The site is just static HTML and CSS, so deployment is fast and maintenance cost is effectively zero. When you want a real docs system later, this can become the landing page in front of Docusaurus, VitePress, DocFX, or MkDocs.

Visual direction

Dark steel + amber accents

Strong contrast, layered gradients, glass panels, and tighter typography.

Quickstart

Enough structure to feel like a framework homepage.

  1. Clone the repo and restore tools.
  2. Build the solution or run the sample host.
  3. Start replacing mock copy with real guides.
dotnet restore InkkSlinger.sln
dotnet build InkkSlinger.sln -c Release -v minimal
dotnet run --project InkkSlinger.csproj
